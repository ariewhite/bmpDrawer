# 

A little console app, for draw .bmp image(only 24 or 32 bit w/ out compression(yet) in console). 

## Install & build
1. clone repo 
> git clone https://github.com/ariewhite/bmpDrawer.git

2. build 
> cd bmpDrawer

> mkdir build && cd build

> cmake .. 

create binaries:

windows:
> cmake --build .

unix*
> make

(remove include windows.h)
